# cyoa-maker
A tool that allows you to build your own Choose Your Own Adventure stories with dynamic elements and world-building tools. Currently built with Node, Express, and the EJS templating language.

Phase 1 - Static Pages:
Get the following functionality built: 
- Create / Delete Pages
- Create / Delete Page Elements; Text, Images, Links to other pages

Phase 2 - Dynamic Pages 
Get the following functionality built:
- Create / Delete Conditions for Page Elements
